Write in Markdown with instant rendering. Toggle between editing and reading modes. Code blocks, lists, quotes, headings — all supported.

  • Full Markdown rendering (markdown-it)
  • Code / Reading mode switch
  • Document outline via headings
  • Minimap for fast scrolling (experimental)
  • Separate EN/CN fonts, adjustable size & line height
